# 🔐 Full-Stack Authentication System

A production-ready full-stack authentication system supporting **JWT-based login** and **OAuth (Google & Facebook)**. Built with modern technologies, this project demonstrates secure user management, scalable backend design, and seamless frontend integration.

---

## 🚀 Live Demo

🌐 https://auth2-0-aeoy.onrender.com   

---

## ✨ Features

- 🔑 User Signup & Login (Email/Password)
- 🔐 Secure Authentication using JWT
- 🔒 Password Hashing with bcrypt
- 🌐 Google OAuth Login
- 📘 Facebook OAuth Login
- 🛡️ Protected Routes (Frontend + Backend)
- ⚡ RESTful API Architecture
- 🌍 Environment-based Configuration
- ☁️ Cloud Deployment (Render)

---

## 🧠 Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)

### Authentication & Security
- JSON Web Tokens (JWT)
- bcrypt
- OAuth 2.0 (Google & Facebook)
